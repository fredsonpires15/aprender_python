## **Visão Geral do Projeto de Gestão Bancária**

- Este projeto visa aplicar os conceitos de programação Python para criar um sistema de gestão bancária que inclui cadastro de clientes, gestão de contas, autenticação e segurança, transações financeiras e gestão de saldo.
    
    ### **1. Cadastro de Clientes**
    
    - **Classe Cliente**
        - **Atributos**:
            - Nome
            - Nº de Identificação
            - Sexo
            - Data de Validade
            - NIF
            - Estado Civil
            - País
            - Distrito
            - Rua
            - Código Postal
            - Data de Nascimento
            - Telemóvel
            - E-mail

        - **Funções**: Cadastro de novos clientes com as informações pessoais fornecidas e guardar num banco de dados.
            - Criar um função chamada **coletar_dados_do_cliente(*self*) que retorna dados do cliente**
                  
            - Criar uma função **banco_de_dados(*self*) que guarda todas as dados do cliente num banco de dados**
            
            - Criar uma função **validar_cliente(*self*) que entra no banco de dados  e verifica o seguinte: Se o cliente existir , ele diz que o cliente já está cadastrado; Se não, ele adiciona o cliente na banco de dados.**
                
    
    ### **2. Gestão de Contas**
    
    - **Classe ContaBancaria**
        - **Atributos**:
            - Número da Conta → num_conta
                
                
            - Tipo de Conta → tipo_conta
                
                
            - Saldo → saldo_inicial
            - Cliente Vinculado → cliente_vinculado
        - **Funções**:
            - Criar novas contas
                
                **Passos para Criar uma Conta**
                
                1. **Obter as informações do cliente**: Certifique-se de que o cliente já esteja cadastrado no sistema.
                2. **Gerar um número de conta único**: Pode ser gerado automaticamente ou fornecido manualmente.
                3. **Definir o tipo de conta**: Escolher o tipo de conta desejada (e.g., Poupança, Corrente).
                4. **Definir o saldo inicial**: Definir o valor inicial a ser depositado.
                5. **Vincular a conta ao cliente**: Associar a conta ao cliente e adicionar a conta à lista de contas do cliente.
            - Vincular contas a clientes
            - Consultar informações das contas

    ### **3. Autenticação e Segurança**
    
    - **Funções**:
        - Registro de clientes permitindo a criação de uma senha
        - Login usando nome de usuário e senha
        - Armazenamento seguro de senhas utilizando técnicas de hash
        
    
    ### **4. Transações Financeiras**
    
    - **Funções**:
        - Realizar depósitos
        - Realizar saques
        - Realizar transferências entre contas
        - Verificar se há saldo suficiente para as transações
    
    ### **5. Gestão de Saldo**
    
    - **Funções**:
        - Atualizar o saldo da conta após cada transação
        - Consultar saldos
    
    ### **6. Gerar IBAN**
    
    - **Função**:
        - Gerar automaticamente um IBAN ao registrar um novo cliente 
        
    
    ### **7. Validar NIF**
    
    - **Função**:
        - Validar automaticamente o NIF ao registrar um novo cliente
